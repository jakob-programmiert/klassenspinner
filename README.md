# Klassenspinner

Eine kleine Browser-App zum Erstellen und Auslosen einer Sitzordnung fuer eine Klasse.

## Funktionen

- Tische und Einzelplaetze im Klassenraum anlegen
- Tische verschieben und drehen
- Sitzplaetze pro Tisch festlegen
- Schuelerinnen und Schueler eintragen
- Freundespaare eintragen, die nicht direkt nebeneinander sitzen sollen
- Sitzordnung per Spin erzeugen
- Klassenstruktur lokal speichern
- Klassenstruktur als JSON-Datei exportieren und in einem anderen Browser wieder laden

## Nutzung

Die App besteht nur aus HTML, CSS und JavaScript. Sie kann direkt ueber `index.html`
geoeffnet oder mit GitHub Pages veroeffentlicht werden.
